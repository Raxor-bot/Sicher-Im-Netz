Grundlegende Konzepte der Internetsicherheit

Was ist Internetsicherheit?

Internetsicherheit bedeutet, dass du dich und deine Daten beim Surfen im Internet schützt. Dazu gehört alles, was du online machst, wie zum Beispiel das Surfen auf Webseiten, das Chatten mit Freunden oder das Einkaufen. Internetsicherheit ist ein Teil von größeren Themen wie der Cybersicherheit, bei der es darum geht, sicherzustellen, dass dein Computer, deine Apps und dein Online-Verhalten sicher sind. [1]

Warum ist Internetsicherheit wichtig?

Wir verbringen viel Zeit im Internet, aber dabei gibt es auch Gefahren. Zum Beispiel könnten Hacker versuchen, auf deine E-Mail oder deine Social-Media-Konten zuzugreifen. Es gibt auch Viren und schädliche Programme, die deinen Computer beschädigen oder deine Daten stehlen könnten. Manchmal versuchen Betrüger auch, deine persönlichen Informationen zu klauen, um sie für kriminelle Zwecke zu nutzen. Internetsicherheit hilft dabei, dich vor diesen Gefahren zu schützen, damit du sicher im Internet unterwegs sein kannst. [1]

Was ist Phishing?

Phishing ist eine Art von Betrug im Internet, bei dem Hacker versuchen, dich mit gefälschten E-Mails hereinzulegen. Diese E-Mails sehen oft so aus, als kämen sie von einer Bank oder einem Kumpel, um dich zu täuschen. Sie wollen, dass du auf einen Link klickst oder einen Anhang öffnest. Wenn du das machst, könnten deine persönlichen Daten gestohlen werden oder dein Computer mit einem Virus infiziert werden. Phishing gibt es schon seit den 1990er Jahren, und es ist immer noch eine beliebte Methode für Betrüger, um an Informationen zu kommen. Heute sind diese Betrugsversuche oft schwerer zu erkennen, weil sie immer besser gemacht werden. [1]

Was ist Malware?

Malware ist eine Mischung aus den englischen Wörtern „malicious“ (was so viel wie „schädlich“ bedeutet) und „Software“. Mit diesem Begriff meint man schädliche Programme wie Viren, Würmer oder Trojaner. Diese Programme werden von Hackern benutzt, um Unordnung zu stiften oder vertrauliche Informationen zu stehlen. Jede Art von Software, die entwickelt wurde, um einen Computer, Server oder ein Netzwerk zu schädigen, nennt man Malware [1]

Was ist eine URL?

Eine URL (das steht für Uniform Resource Locator) ist wie eine Adresse im Internet. Sie zeigt genau, wo eine Datei oder eine Webseite auf einem Server gespeichert ist. Wenn du eine URL in deinen Webbrowser eingibst, gelangst du zu der Webseite oder Datei, die dort gespeichert ist. URLs können auch in einem Text eingebaut werden, damit man durch Anklicken direkt zu einer anderen Webseite kommt. [2]

Quelle: https://www.eology.de/wiki/url

Wie erkenne ich, ob eine Website sicher ist?

Es gibt Hinweise, die dir helfen können, eine sichere Website zu erkennen. Das wichtigste Zeichen ist das Übertragungsprotokoll am Anfang der URL:

  • HTTPS: Sicher. Deine Daten werden verschlüsselt zwischen deinem Computer und dem Server übertragen. Das bedeutet, dass niemand, der die Verbindung abhört, deine Daten einfach lesen kann.
  • HTTP: Unsicher. Deine Daten werden unverschlüsselt übertragen, was bedeutet, dass sie auf dem Weg vom Nutzer zum Server abgefangen und gelesen werden könnten.

Kurz gesagt: HTTP ist unsicher, HTTPS ist sicher. Wenn du sensible Informationen wie Passwörter oder Kreditkartendaten eingibst, solltest du immer darauf achten, dass die Website HTTPS verwendet.


Tausch dich nun mit deinem Partner über diese Fragen aus:

  • Warum ist es sicherer, eine Website mit HTTPS statt HTTP zu verwenden?

    • Lösung: Daten werden verschlüsselt übertragen

  • In welchen Fällen reicht http aus? (Eingabe von Daten)

    • Lösung: Wenn kene Daten übertragen werden

  • Wie kannst du überprüfen, ob eine kommunikation sicher (HTTPS) ist, bevor du persönliche Informationen eingibst?

    • Lösung: auf die URL im Browserfenster klicken oder auf den Knopf links davon

  • Schau dir die URL dieser Website an: Nutzt sie HTTPS oder HTTP?

    • Lösung: https

Was ist eine Domain?

Eine Domain ist ein einzigartiger Name, der einer bestimmten Website im Internet gehört. Jede Website hat ihre eigene, einmalige Domain, damit du sie leicht finden kannst. Ein Beispiel für eine Domain wäre "www.beispielseite.com". [4]

Betrüger versuchen manchmal, echte Domains nachzuahmen, um dich auf falsche Websites zu locken und so an deine Daten zu gelangen. Deshalb ist es wichtig, immer genau auf den Domainnamen zu achten. Dafür jetzt ein Beispiel.


Beispiele für echte und gefälschte URLs:

Echte URL: https://www.instagram.com/

Gefälschte URL: https://www.lnstagram.com/ (beachte das kleine „L“ das im Browser wie ein großes "i" aussieht)

Funfact: Die URL "www.lnstagram.com" (mit einem kleinen "L" statt einem großen "i") führt tatsächlich zur echten Instagram-Seite. Instagram hat sich diese URL gesichert, um zu verhindern, dass Betrüger sie nutzen können. Dies ist ein cleverer Schutzmechanismus, um dich und andere vor möglichen Phishing-Angriffen zu schützen

Gefälschte URL: https://www.1nstagram.com/ (beachte die „1“ statt „i“)

Es gibt viele Möglichkeiten, wie Betrüger versuchen, dich zu täuschen. Sei immer wachsam!

Wichtige Punkte zu Cookies:

  • Cookies speichern Informationen über dich, z.B. deine Login-Daten oder deine Präferenzen auf einer Website.
  • Manche Cookies, wie Tracking-Cookies, können verwendet werden, um dein Verhalten im Internet nachzuverfolgen.
  • Es ist wichtig, deine Cookie-Einstellungen zu überprüfen, um deine Privatsphäre zu schützen.

Wie kannst du deine Cookie-Einstellungen überprüfen?

Hier ist eine einfache Anleitung, wie du deine Cookie-Einstellungen in den gängigsten Browsern überprüfen und anpassen kannst:

  • Google Chrome: Gehe zu den Einstellungen > Datenschutz und Sicherheit > Website-Einstellungen > Cookies und Websitedaten.
  • Mozilla Firefox: Gehe zu den Einstellungen > Datenschutz & Sicherheit > Cookies und Websitedaten.
  • Microsoft Edge: Gehe zu den Einstellungen > Website-Berechtigungen > Cookies und Websitedaten.
